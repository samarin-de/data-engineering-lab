# Data Engineering Lab

Мои учебные проекты по Data Engineering.

## Стек технологий
- **Apache Airflow** — оркестрация пайплайнов
- **Apache Spark (PySpark)** — обработка больших данных
- **PostgreSQL** — хранение данных
- **Docker** — контейнеризация
- **Python** — скрипты и трансформации

## Содержание
- `dags/` — все DAG-файлы для Airflow
- `spark_etl.py` — пример ETL на PySpark

## Как запустить
1. Склонировать репозиторий
2. Поднять окружение через `docker-compose up -d`
3. Зайти в Airflow `localhost:8080`
